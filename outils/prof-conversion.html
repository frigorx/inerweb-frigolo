💡 Notes pédagogiques

  • Durée estimée : 45 min à 1h15 selon le niveau
  • Progression : chaque phase doit être validée à 60% pour débloquer la suivante
  • Aléatoire : les exercices changent à chaque chargement — pas de copie possible entre élèves
  • Reprise : la progression est sauvegardée dans le navigateur (même téléphone = reprise automatique)
  • Résultats : envoyés au Google Sheet uniquement à la Phase 7 (évaluation finale)
  • Hors ligne : le module fonctionne sans connexion (sauf l'envoi des résultats, qui sera mis en file d'attente)
  • Tolérance : conversions ±0,05 mm / développés ±0,5 mm / pouces décimaux ±0,005
