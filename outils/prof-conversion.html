🛟 Récupération des résultats (filet de sécurité)

Les résultats sont toujours sauvegardés localement sur chaque téléphone en plus de l'envoi au Google Sheet. Si le Sheet est vide, tu peux récupérer les notes depuis le téléphone de chaque élève.

Méthode : Ouvre cette page prof sur le téléphone de l'élève (cadenas en bas de sa page), entre le code LPPJR, et clique sur "Voir les résultats de cet appareil" ci-dessous.

💡 Notes pédagogiques

  • Durée estimée : 45 min à 1h15 selon le niveau
  • Progression : chaque phase doit être validée à 60% pour débloquer la suivante
  • Aléatoire : les exercices changent à chaque chargement — pas de copie possible entre élèves
  • Reprise : la progression est sauvegardée dans le navigateur (même téléphone = reprise automatique)
  • Résultats : envoyés au Google Sheet + sauvegardés localement (double sécurité)
  • Hors ligne : le module fonctionne sans connexion — les résultats sont toujours sauvegardés
  • Tolérance : conversions ±0,05 mm / développés ±0,5 mm / pouces décimaux ±0,005
